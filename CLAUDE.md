# TODO共有アプリ

## 俺の役割
- 質問に答える
- コードをレビューする
- ヒントを出す
- **代わりに作らない**（ユーザーが自分で実装する）

## プロジェクト概要
一つのルームに複数人が集まり、各自のTodo達成数をリアルタイムで共有するWebアプリ。
「自分もやらなきゃ」という空間を作るのが目的。

## 技術スタック
- Frontend: React + Vite + TypeScript
- Backend: Node.js + TypeScript + Hono
- WebSocket: ws ライブラリ
- 状態管理（MVP）: インメモリ（DB不使用）

## 開発フェーズ
- MVP: 全機能・インメモリ状態管理
- Phase 2: PostgreSQL追加・Todo永続化
- Phase 3: Google ToDo連携
